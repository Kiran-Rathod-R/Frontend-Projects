# 🧮 Calculator

A responsive calculator application built using HTML, CSS, and JavaScript that performs basic arithmetic operations through an intuitive user interface.

## 📖 Overview

This project demonstrates JavaScript event handling, DOM manipulation, and mathematical operations. It provides a simple and interactive calculator experience for users.

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear and Delete functionality
- Responsive interface

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Future Improvements

- Scientific calculator
- Keyboard support
- Calculation history
- Theme switcher
