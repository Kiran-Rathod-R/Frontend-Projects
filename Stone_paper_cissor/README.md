# ✂️ Stone Paper Scissors

A browser-based implementation of the classic Rock Paper Scissors game using JavaScript.

## 📖 Overview

The project demonstrates JavaScript logic, random number generation, event handling, and score tracking through an interactive game.

## ✨ Features

- Player vs Computer
- Random computer moves
- Live score tracking
- Winner announcement
- Responsive interface

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript

## 🚀 Future Improvements

- Multiplayer mode
- Game statistics
- Sound effects
- Difficulty levels
