About Smart Stationary

📌 Project Overview

Smart Stationary is a web-based shopping application designed to simulate an online stationery store. Users can browse stationery items, add them to a cart, and place orders easily.

⚙️ Features

  • Browse stationery products
  • Add items to shopping cart
  • Dynamic total price calculation
  • Admin dashboard to add new products
  • Simple and responsive user interface

💻 Technologies Used

  • HTML – Structure of the website
  • CSS – Styling and layout
  • JavaScript – Dynamic functionality

🎯 Project Purpose

The goal of this project is to understand how a basic e-commerce system works including product display, cart management, and simple admin controls.
