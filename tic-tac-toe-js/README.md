# ❌⭕ Tic Tac Toe

A classic Tic Tac Toe game built with HTML, CSS, and JavaScript featuring an interactive game board and winner detection.

## 📖 Overview

This project demonstrates JavaScript game logic, DOM manipulation, and responsive interface design through an engaging browser game.

## ✨ Features

- Two-player gameplay
- Winner detection
- Draw detection
- Restart game
- Responsive layout

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript

## 🚀 Future Improvements

- AI opponent
- Online multiplayer
- Scoreboard
- Sound effects
